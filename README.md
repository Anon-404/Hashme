## Installation

For debian based linux os

```bash
sudo apt install python3 -y
git clone https://github.com/MRZ724/Hashme
cd Hashme
sudo pip3 install hashlib
python3 Hashme.py
```
For termux

```bash
pkg install python -y
pkg install git -y
git clone https://github.com/MRZ724/Hashme
cd Hashme
pip install hashlib
python Hashme.py
```

For arch based linux os

```bash
sudo pacman -Sy python3 -y
git clone https://github.com/MRZ724/Hashme
cd Hashme
sudo pip3 install hashlib
python3 Hashme.py
```
